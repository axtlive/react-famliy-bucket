# react-redux

- React: 组件化的UI界面处理方案
- React-Router：根据地质匹配路由，最终渲染不同的组件
- Redux：处理数据以及数据变化的方案（主要用于处理共享数据）

> 入股go一个组件，仅用于渲染一个UI界面，而没有状态（通常是一个函数组件），改组件叫做**展示组件** 
> 如果一个组件，仅用于提供数据，没有任何属于自己的UI界面，则该组件叫做**容器组件**，容器组件纯粹是为了给其他组件提供数据
